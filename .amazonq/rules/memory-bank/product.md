# 제품 개요

## 프로젝트 목적
Progressive Web App(PWA)으로 구축된 정교한 작업 관리 및 시간 추적 애플리케이션. 계획 → 실행 → 검토의 3단계 워크플로우를 구현하여 집중적인 생산성과 상세한 성과 분석을 위해 설계됨.

## 주요 기능

### 핵심 기능
- **3단계 워크플로우**: FOCUS(작업 실행) → SUMMARY(결과 검토) → CALENDAR(기록 데이터)
- **고급 시간 추적**: 분/초 정밀도의 실시간 타이머, 백그라운드 작동 지원
- **진행률 관리**: 개별 작업 진행률 추적 및 퍼센트 완료도
- **드래그 앤 드롭 인터페이스**: 터치 최적화된 정렬 가능한 작업 목록
- **스마트 자동완성**: 과거 데이터 기반 작업명, 시간, 진행률 제안

### 데이터 관리
- **이중 저장소**: 로컬 IndexedDB와 선택적 Supabase 클라우드 동기화
- **기록 로깅**: 성과 지표가 포함된 완전한 일일 작업 기록
- **캘린더 통합**: 일별 완료 통계가 있는 시각적 캘린더
- **데이터 지속성**: 오프라인 우선 아키텍처로 자동 저장

### 사용자 경험
- **Progressive Web App**: 설치 가능하고 오프라인 지원하는 모바일 경험
- **반응형 디자인**: 모바일 우선 접근 방식의 터치 최적화 인터페이스
- **시각적 피드백**: 애니메이션 전환 및 상태 표시기
- **접근성**: 키보드 탐색 및 스크린 리더 지원

## 대상 사용자
- **생산성 애호가**: 상세한 작업 추적 및 성과 분석을 원하는 개인
- **시간 관리 전문가**: 정밀한 시간 로깅 및 기록 보고가 필요한 사용자
- **모바일 워커**: 오프라인 지원 작업 관리가 필요한 전문가
- **데이터 중심 사용자**: 지표와 기록 성과 추적을 중시하는 사람들

## 사용 사례
- 일일 작업 계획 및 실행 추적
- 과거 데이터를 통한 시간 추정 정확도 개선
- 여러 날에 걸친 성과 패턴 분석
- 모바일 우선 생산성 관리
- 동기화 기능이 있는 오프라인 작업 관리
